CSGPlot V1.0 使用手册

COMTRADE 波形查看器 — 面向电力系统暂态分析的桌面工具  |  作者：CSG Huang Yihong

1. 简介与系统要求

CSGPlot 是一款 Windows 桌面应用，用于读取 IEEE COMTRADE（1999/2013）波形记录文件，在交互式多子图界面中展示模拟量和数字量通道。支持公式编辑器生成计算波形、FFT 频谱分析、以及矢量图形导出，适用于故障分析、继电保护测试复核和技术报告制作。

支持格式 COMTRADE ASCII / Binary（.cfg + .dat + .dmf/.hdr），CSV/TXT
操作系统 Windows 10 / 11 (x64)
运行环境 Python 3.11–3.13（从源码运行）或直接双击 exe
依赖库 PySide6 ≥ 6.7, pyqtgraph ≥ 0.13.7, numpy ≥ 1.26, pywin32
首次使用建议先阅读 界面布局 和 通道拖放 两节，这是最核心的操作流程。

2. 启动与文件加载

2.1 启动方式

  • 双击 exe：直接运行 CSGPlotV1.0.exe，无任何依赖。
  • 命令行传参：在 PowerShell 中执行 .\CSGPlotV1.0.exe .\recording.CFG 可直接在启动时打开指定波形文件，支持传入多个文件。

2.2 加载 COMTRADE 文件

CSGPlot 支持三种加载文件的方式：

  1. Open 按钮：点击顶部工具栏的 Open 按钮（或按 Ctrl+O），在文件对话框中选中 .cfg 或 .txt 文件。软件会自动寻找同名 .dat 文件；对于二进制格式还会读取 .dmf 和 .hdr。
  2. 拖拽文件：将 .cfg 或 .txt 文件从资源管理器直接拖入左侧 Documents 区域，软件自动加载。可一次拖入多个文件。
  3. 命令行启动：启动时在命令行中传入文件路径（见 2.1）。
加载大文件时使用后台线程，窗口不会冻结。状态栏会显示加载进度。若加载失败，将弹出错误提示对话框。
当前版本支持 ASCII 和 BINARY 两种 COMTRADE DAT 格式。文件对话框虽列出 .cff，但暂不支持该格式。

3. 界面布局

主窗口分为四大区域：

① 顶部工具栏 Open、Export、Calculate、FFT、Max/Min、Add/Remove Plot、对齐控件、AB Data
② 左侧边栏 上半部分 Documents（已加载文件列表），下半部分 Channels（当前选中文件的通道列表），以及 计算结果 面板
③ 波形子图区 默认 5 个空白子图，最多 12 个。每个子图左侧有彩色波形标签栏
④ 底部状态栏 显示就绪状态、A/B 标记线时间差（dt）与频率（f），右下角显示开发者署名

4. 左侧面板

4.1 Documents 文件列表

已加载的每个 COMTRADE 文件作为一个独立 Document 显示在左侧上半区域。文件列表条目格式为：

文件名 (N个模拟量通道A, M个数字量通道D)

  • 单击某个 Document 条目，右侧 Channels 面板会切换显示该文件的通道列表。
  • 同一个会话中可同时打开多个 CFG 文件，来自不同文件的通道可以拖入同一个子图中进行对比。
  • 选中 Document 后，其通道颜色按预设调色板自动分配（8 种颜色循环）。

4.2 Channels 通道列表

选中 Document 后，该文件的所有模拟量通道（Analog）以彩色标签形式显示。每个标签格式为：

ch_name (phase) [unit]

  • 拖放：用鼠标按住通道标签，直接拖入右侧任意子图即可显示波形。详见 通道拖放。
  • 单位显示：当数据源提供了通道单位时，标签中会展示。若子图中所有可见波形共享同一单位，Y 轴自动标注该单位。

4.3 计算结果面板

通过 Calculate 功能创建的计算波形会出现在侧边栏底部的「计算结果」面板中，同样可拖入子图。每条计算结果旁有 X 按钮可删除。

5. 顶部工具栏

工具栏分为 5 个功能组，从左至右：

File 组

  • Open：打开 COMTRADE 文件（Ctrl+O）。
  • Export：弹出导出对话框，可勾选子图后导出为 PNG、SVG 或复制 EMF 到剪贴板。

Analyze 组

  • Calculate：打开公式编辑器，创建计算波形（支持 +、-、*、/、括号、系数、内置函数）。
  • FFT：对选中通道执行 FFT 频谱分析，进入双窗口（时域 + 频域）视图。
  • Max / Min：可勾选的按钮，激活后在 A/B 标记线之间显示各波形的最大值或最小值标记。

Layout 组

  • Add Plot：新增一个空白子图（最多 12 个）。
  • Remove Plot：删除当前选中的子图。

View Align 组

包含时间轴对齐下拉框和波形对齐控件，详见 时间对齐。

Markers 组

  • AB Data：可勾选按钮。激活后在每个子图中显示 A/B 标记线对应 Y 轴数据。详见 A/B 标记线。

6. 子图管理

子图是 CSGPlot 显示波形的基本单元：

  • 默认数量：启动时自动创建 5 个空白子图。
  • 数量范围：最少 1 个，最多 12 个。点击 Add Plot 新增，点击 Remove Plot 删除当前选中子图。
  • 选中子图：单击子图区域（非波形标签区域）即可选中。当前选中的子图边框高亮。部分快捷键（如 Y）仅作用于当前选中子图。
  • 鼠标滚轮：在波形区域滚动鼠标滚轮会垂直滚动子图列表，而非缩放波形。这一设计避免了误操作。
  • 删除波形：双击子图左侧的彩色波形标签即可从该子图中移除该波形。

7. 通道拖放

通道拖放是 CSGPlot 最核心的交互方式。操作流程如下：

  1. 在左侧 Channels 列表中，用鼠标左键按住一个通道标签。
  2. 拖动到右侧目标子图的波形区域上方。
  3. 释放鼠标，波形即在该子图中渲染。

拖放规则与高级操作：

  • 跨文件拖放：可将不同 Document 的通道拖入同一子图，实现不同录波文件的波形对比。
  • 子图间拖放：拖动子图左侧的彩色波形标签到另一个子图，会在目标子图中复制一份该波形视图（原波形保留）。
  • 波形标签颜色：每个通道的颜色在会话中保持一致，便于跨子图识别同一物理量。
  • Y 轴单位：当一个子图中所有可见波形共享同一个单位时，Y 轴自动标注该单位；单位不一致时 Y 轴不显示单位。

波形标签交互：

  • 单击标签：将该波形设为当前子图的"活跃 Y 轴波形"，Y 轴范围自动适配该波形，并在 Y 轴上显示其单位。
  • Ctrl+单击标签：将该波形加入"对齐选中集"，用于后续执行 Start / Mutation 对齐操作。
  • 双击标签：从子图中移除该波形。
  • 拖拽标签：将波形复制到另一个子图。

8. 导航与缩放

8.1 键盘快捷键

  • X — 自动适配全局 X 轴：根据当前子图所有波形的数据范围，统一设置所有子图的 X 轴（时间轴）范围。
  • Y — 自动适配当前子图 Y 轴：根据当前子图当前可视 X 范围内所有波形的 Y 值，自动设置该子图的 Y 轴范围。

8.2 鼠标拖拽缩放

在子图波形区域按住鼠标左键并拖拽，软件会根据拖拽方向自动判断缩放模式：

  • 水平拖拽（宽 > 3× 高）：仅缩放 X 轴（时间轴），Y 轴保持不变。所有子图的 X 轴联动。
  • 垂直拖拽（高 > 3× 宽）：仅缩放 Y 轴，X 轴保持不变。仅作用于当前子图。
  • 矩形拖拽（其他比例）：同时缩放 X 和 Y 轴。X 轴全局联动，Y 轴仅作用于当前子图。
拖拽距离过小（< 1px）不会触发缩放，避免单击时误缩放。

8.3 精确设置轴范围

  • 双击 X 轴区域（子图底部 36px 范围内），弹出对话框输入 X 轴最小值和最大值（逗号分隔）。
  • 双击 Y 轴区域（子图左侧 58px 范围内），弹出对话框输入 Y 轴最小值和最大值。

8.4 智能时间轴

X 轴自动在毫秒（ms）和秒（s）之间切换：

  • 当可视范围内最大刻度 > 100 ms 时，显示为秒（s）。
  • 否则显示为毫秒（ms）。
  • 小数位数根据刻度步长自动确定，保证至少两位有效数字。

9. 时间对齐

9.1 全局时间参考（Alignment 下拉框）

位于工具栏 View Align 组上方，提供三种时间对齐模式，对所有子图生效：

  • Trigger（触发时刻）：以录波文件的触发时刻（trigger_time）作为时间零点。各文件的触发时刻对齐到 t=0。
  • Start（起始时刻）：以录波文件的第一个采样点作为时间零点。
  • Absolute（绝对时间）：以实际时间戳（start_time）为基准，不同文件按绝对时间对齐。

9.2 波形级时间对齐

工具栏 View Align 组提供精细的波形对齐控件：

自动对齐

  • Start 按钮：将对齐选中的波形，使其第一个采样点的时间对齐到基准波形的起始时间。
  • Mutation 按钮：将对齐选中的波形，使其最大突变点（导数绝对值最大处）对齐到基准波形的对应位置。

手动偏移

  • 在输入框中键入偏移值（单位：ms），点击 Offset 按钮对选中的波形应用该时间偏移。
  • 点击 Clear 按钮清除选中波形的时间偏移，恢复默认对齐。

如何选中波形进行对齐

  1. 子图级选中：单击子图选中它，然后使用对齐按钮 — 操作将作用于该子图中所有波形。
  2. 波形级选中：Ctrl + 单击子图的波形标签，将该波形加入对齐选中集。之后的对齐操作仅作用于选中的波形。

10. A/B 标记线

10.1 显示与隐藏

  • 按 A 键切换 A 标记线（红色）的显示/隐藏。
  • 按 B 键切换 B 标记线（蓝色）的显示/隐藏。

首次按下时，标记线会出现在鼠标光标所在子图的对应位置。如果光标不在任何子图上，标记线出现在当前选中子图的 X 轴中点。

10.2 拖动标记线

用鼠标左键按住标记线附近（容差 18px），直接拖动到目标时间位置。拖动时标记线实时跟随鼠标。

10.3 标记线数据 (AB Data)

点击工具栏 AB Data 按钮激活后：

  • 每个子图中显示 A 和 B 标记线位置各波形的 Y 轴数值。
  • 底部状态栏显示 dt（A、B 线之间的时间差，单位 ms）和 f（对应频率 = 1/dt）。
  • 再次点击 AB Data 按钮关闭数值显示。

11. 最大值 / 最小值标记

工具栏上的 Max 和 Min 按钮是可勾选的开关：

  • Max：激活后，在各子图的 A/B 标记线之间的区间内，为每条波形标注其最大值点。
  • Min：激活后，同样在 A/B 区间内标注各波形的最小值点。
  • 两个按钮可独立勾选，也可同时勾选。
  • 如果 A 或 B 标记线未激活，按钮仍可勾选但不会显示标记（因为没有有效区间）。

12. 计算波形

点击工具栏 Calculate 按钮打开公式编辑器对话框。计算波形是由已加载通道通过数学表达式合成的新波形。

12.1 公式语法

  • 变量：使用 S1、S2、S3…… 表示已加载的通道。公式编辑器对话框中会列出所有可用变量及其对应的通道名称。
  • 运算符：+ - * / ()，支持括号嵌套。
  • 系数：直接写数字，如 2*S1、0.5*S2。
  • 内置函数（从对话框中选择）：
    • abs(x) — 绝对值
    • integral(x) — 积分（对时间）
    • diff(x) — 微分（对时间）

12.2 示例

公式 含义
S1 + S2 S1 与 S2 的逐点相加
2*S1 - S2 S1 的两倍减去 S2
integral(S1) S1 对时间的积分
abs(S1 - S2) S1 与 S2 之差的绝对值

12.3 时间轴插值

当公式中的变量使用不同的时间轴（来自不同采样率的文件或对齐全后偏移的波形）时，非基准波形会被自动线性插值到基准波形的时间轴上，确保逐点运算的正确性。

12.4 使用计算结果

创建成功后，计算结果出现在左侧边栏底部的「计算结果」面板中。该结果可以像普通通道一样拖入任意子图显示。点击结果旁的 X 按钮可删除。

13. FFT 频谱分析

在左侧 Channels 列表中选中一个通道，然后点击工具栏 FFT 按钮（或通过通道右键菜单），打开 FFT 分析对话框。

13.1 分析窗口

FFT 对话框分为上下两个图：

  • 上方 Signal 图：时域波形。其中高亮区域为 FFT 分析窗口。可用鼠标拖拽窗口边界来调整分析范围。
  • 下方 FFT 图：频域幅度谱。X 轴可选择频率（Hz）或谐波次数（Harmonic），Y 轴可选择多种显示方式。

13.2 参数设置

基频 (Hz) 电力系统基频，默认 50 Hz
周期数 分析窗口包含的基频周期数
起始时间 (s) 分析窗口的起始时刻
最大频率 (Hz) 频谱显示的上限频率
窗函数 Hanning / Hamming / Blackman / Rectangular / Bartlett / Kaiser
去直流 勾选后从信号中减去均值后再做 FFT

13.3 显示模式

  • Bar — Absolute：柱状图，显示绝对幅度值。
  • Bar — % Fundamental：柱状图，以基频幅度为 100% 的百分比显示。
  • Bar — % Base Value：柱状图，以自定义基准值为 100% 的百分比显示。
  • Bar — % DC：柱状图，以直流分量为 100% 的百分比显示。
  • List — *：上述四种模式的文本列表版本，显示各次谐波的频率、幅度（或百分比）和相位角。

13.4 谐波列表

选择 List 显示模式时，对话框右侧或底部显示详细的谐波分析列表，包括：

  • 采样时间 (dt)
  • 每周期采样点数
  • 直流分量
  • 基频幅值（峰值 / RMS）
  • 总谐波失真 THD (%)
  • 各次谐波的频率、幅度、相位角

13.5 导出 FFT 图形

右键点击 FFT 对话框中的任意图形区域，可导出为 PNG 或复制 EMF 到剪贴板。

14. 导出与剪贴板

14.1 工具栏 Export 按钮

点击 Export 按钮弹出导出对话框：

  • 勾选要导出的子图（多选）。
  • 选择导出格式：PNG（位图）、SVG（简化矢量图）、EMF（复制到剪贴板）。
  • 可设置导出宽度（pt），留空则使用默认宽度 900pt。

14.2 单子图右键导出

在任意子图上右键点击，弹出上下文菜单：

  • Export as PNG：将当前子图导出为 PNG 文件。PNG 最小宽度 2048px，96 DPI。
  • Export as SVG：将当前子图导出为 SVG 文件。SVG 包含轴、网格、标签和波形多段线。
  • Copy as EMF：将当前子图以 EMF 矢量格式复制到系统剪贴板，可直接粘贴到 Microsoft Visio 中作为可编辑的矢量图形。

14.3 导出文件说明

PNG 位图格式，适合快速分享。字体 Segoe UI，刻度 9pt，标签 10pt。
SVG 简化矢量图，仅包含关键元素（坐标轴、网格、标签、波形线），文件体积小，适合嵌入文档。
EMF Windows 增强图元文件，通过剪贴板传递。专门为粘贴到 Microsoft Visio 设计，粘贴后为可编辑矢量内容。

15. 粘贴波形数据

CSGPlot 支持从剪贴板粘贴波形数据，用于快速查看外部数据（如从 Excel 复制的数据）。

15.1 数据格式

剪贴板数据必须是两列或多列 CSV 文本：

  • 第一行为列标题（逗号分隔），第一列为时间轴（任意名称，单位视为秒），其余列为模拟量通道名称。
  • 第二行起为数值数据，逗号分隔。

示例：

Time, Voltage_A, Current_A
0.000, 0.00, 0.01
0.001, 0.15, 0.03
0.002, 0.32, 0.08
...

15.2 粘贴操作

  1. 复制波形数据到剪贴板（Ctrl+C）。
  2. 在 CSGPlot 中选中一个子图（或确保有子图存在）。
  3. 按 Ctrl+V。
  4. 软件自动解析剪贴板内容，创建一个虚拟 Document（命名 Clip_N），并将所有通道添加到选中子图中。
粘贴的数据仅在当前会话中保留，关闭 CSGPlot 后即丢弃。如需长期保存，请将数据导出为 COMTRADE 格式文件。
