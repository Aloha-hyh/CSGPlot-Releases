# CSGPlot

CSGPlot 是一款专为电力系统仿真数据处理设计的工具。主要用于实现仿真结果的快速可视化与数据分析，提升科研绘图效率。

## 核心功能
* **多格式兼容**：支持主流仿真软件输出的数据格式。
* **自动化绘图**：提供标准化的坐标轴处理与曲线样式配置。
* **数据分析**：内置电力系统常用指标计算（如暂态分析、频谱分析等）。
